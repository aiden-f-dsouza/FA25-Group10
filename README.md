## Project Overview

UIUC TurboLearn is a comprehensive student resource platform designed to enhance learning through collaborative content sharing and AI-powered study tools. Students can upload notes, questions, and resources for their classes, and the platform intelligently processes existing resources to generate additional study materials.

## Core Features

- **Resource Upload**: Students can upload notes, questions, and study resources for specific classes
- **Web Scraping Integration**: Automatically scrapes and processes subject-specific resources from the web
- **AI-Powered Content Generation**: Leverages AI to create study notes and flashcards from existing resources
- **Class Organization**: Organize and access resources by course and subject matter

## Technology Stack

- Frontend: JavaScript (for seamless Supabase deployment)
- Backend: Supabase integration and FastAPI 
- AI/ML: Content processing and generation
- Web Scraping: Automated resource collection
